# Phase 0 — Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up Next.js 15 + Payload CMS 3 + Postgres + Vercel with admin auth, CI, and zero application features. Every later phase assumes this foundation without re-justifying it.

**Architecture:** A single Next.js App Router project with Payload installed in-process. Postgres runs in Docker locally and on Neon in production. Vercel hosts both the front-end and Payload's admin/API. Media uploads go to Vercel Blob.

**Tech Stack:** Next.js 15, React 19, TypeScript 5.4+ (strict), Payload CMS 3, Postgres (via Docker locally, Neon in prod), pnpm, Vitest, Playwright, Husky, lint-staged, commitlint, GitHub Actions, Vercel.

**Spec reference:** [2026-04-17-pen-and-paper-design.md §10, §13](../specs/2026-04-17-pen-and-paper-design.md)

    await page.goto('/admin');
    await expect(page.getByLabel(/email/i)).toBeVisible();
    await expect(page.getByLabel(/password/i)).toBeVisible();

});
